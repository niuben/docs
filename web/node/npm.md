## npm

#### npm init
创建package.json文件

#### npm install 
安装模块

#### npm update

#### npm publish
发布模块

#### npm adduser

#### npm link
将模块命令设置成全局命令
操作步骤
* 在模块下创建bin文件夹，编写命令和对应文件路径
* 执行npm link
